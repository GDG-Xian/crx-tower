Tower.im Plus
===============

Tower.im 的 Chrome 增强扩展，增强其富文本编辑器的功能

目前实现了以下的扩展功能：

- 添加插入链接的功能
- 添加缩进和取消缩进的功能
- 为讨论的编辑器中也添加水平分豁线的功能
